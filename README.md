# Career DNA Analyzer

**Description:**  
A full-stack recommendation system that maps user skills to suitable career paths. The system uses AI APIs to generate answers and provide personalized career suggestions, combining both back-end logic and front-end interface.

**Features:**  
- Collects and structures user skill data  
- Uses AI API to generate personalized career recommendations  
- Displays results via a web interface for better usability  
- Implements full-stack architecture with clean data handling

- Front-end: HTML, CSS, JavaScript

- AI Integration: API-based answer generation
